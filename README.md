# openclaw-docs-mirror

Public mirror workspace for an official-docs-first OpenClaw workflow.

## Purpose
- Keep a clean reference path for technical help.
- Prefer official OpenClaw docs/repo guidance over random web content.
- Reduce conflicting setup advice.

## Automation
This repo is updated by a scheduled OpenClaw sync flow.

See [`AUTOMATION.md`](./AUTOMATION.md) for details:
- why this exists
- what gets synced
- update cadence
- auth requirements
- troubleshooting

## Security note
This public repo should not contain machine-specific sensitive details
(e.g., absolute local storage paths, personal tokens, private hostnames).
